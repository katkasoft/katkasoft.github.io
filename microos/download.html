Installation Guide

1. Create Bootable USB

Linux/macOS:

# Find your USB device
lsblk

# Write ISO (REPLACE /dev/sdX with your device!)
sudo dd if=microos-v0.1.iso of=/dev/sdX bs=4M status=progress
sync

WARNING: Replace /dev/sdX with your USB device (e.g., /dev/sdb). Wrong device will ERASE YOUR DATA!

Windows:

Use one of these tools:

  • Rufus (Recommended)
  • Balena Etcher (User-friendly)
  • Win32 Disk Imager

2. Boot from USB

Restart your computer and:

  • Press F12, F2, ESC, or DEL during startup
  • Select your USB drive from boot menu
  • If it doesn't boot, check BIOS/UEFI settings:
    • Enable "Legacy Boot" or "CSM" if available
    • Disable "Secure Boot"
    • Set USB as first boot device

3. Using MicroOS

MicroOS will boot directly to the terminal interface in about 5-10 seconds. No installation required!

First Steps:

  1. Type help or commands to see available commands
  2. Use sd to list directory contents
  3. Use os-info to see system information
  4. Use sys-stat to check resource usage
  5. Type poweroff to shut down

4. Persistent Storage (Optional)

For saving files across reboots:

On Linux:

# After creating bootable USB
sudo cfdisk /dev/sdX
# Create new partition, type: Linux (83)
sudo mkfs.ext4 /dev/sdX2 -L MICROOS_DATA

Troubleshooting

Won't Boot

Try different USB port (USB 2.0 often works better). Disable Secure Boot in BIOS.

Keyboard Not Working

Try PS/2 keyboard if available. USB keyboards may need different port.

Black/Blank Screen

Try adding nomodeset to kernel parameters in GRUB (press 'e' at boot).

Kernel Panic

Check ISO integrity with SHA256. Try on different hardware.
