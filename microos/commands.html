MicroOS Command Reference

Complete documentation for all available commands in MicroOS v0.1

Type help or commands in the MicroOS terminal to see this list.

Command Categories

  • Terminal Commands
  • System Commands
  • Information Commands
  • File Operations
  • Directory Operations
  • Process Commands

Terminal Commands

clear

clear

Clears the terminal screen.

Example: clear

exit

exit

Exits the terminal session. This will shut down MicroOS as the terminal is the main process.

Example: exit

pcd

pcd

Prints the current working directory.

Example: pcd Output: /home/user

gtd

gtd [path]

Changes the current directory to the specified path.

Example: gtd /tmp Output: Changes directory to /tmp

System Commands

poweroff

poweroff

Safely powers off the computer. Calls sync() to flush filesystem caches before shutdown.

Example: poweroff

restart

restart

Restarts the computer.

Example: restart

mount

mount [device] [mount_point]

Mounts a filesystem. Currently supports ext4 filesystems.

Example: mount /dev/sda1 /mnt Output: Mounts /dev/sda1 at /mnt

Information Commands

sys-stat

sys-stat [options]

Displays system resource usage statistics.

Options:
  • -r, --ram: Show only RAM usage
  • -c, --cpu: Show only CPU usage
  • -d, --disk: Show only disk usage
  • -u, --uptime: Show only system uptime
  • --help: Display help information
Example: sys-stat -c Output: 45%

os-info

os-info [options]

Displays operating system and kernel information.

Options:
  • -v, --version: Show only MicroOS version
  • -k, --kernel: Show only kernel information
  • --help: Display help information
Example: os-info -k Output: Linux 6.18.5-arch1-1

hw-info

hw-info [options]

Displays hardware information.

Options:
  • -c, --cpu: Show only CPU model
  • -r, --ram: Show only total RAM
  • -d, --disk: Show only total disk capacity
  • --help: Display help information
Example: hw-info -c Output: Intel(R) Core(TM) i5-XXXX CPU @ X.XXGHz

File Operations

create file

create file [filename]

Creates a new empty file.

Example: create file document.txt

read

read [filename]

Displays the contents of a file.

Example: read document.txt

write

write [filename] [options]

Interactive file editor. Type 'q' to exit.

Options:
  • -a, --append: Append to file instead of overwriting
  • --help: Display help information
Example: write notes.txt -a

copy

copy [filename]

Copies a file to the clipboard (temporary buffer).

Example: copy document.txt

paste

paste [destination_path]

Pastes a file from clipboard to specified location.

Example: paste /home/user/

cut

cut [filename]

Cuts a file (copies to clipboard and deletes original).

Example: cut document.txt

rename

rename [oldname] [newname]

Renames a file.

Example: rename old.txt new.txt

del

del [filename]

Deletes a file.

Example: del temporary.txt

moveto

moveto [file] [destination]

Moves a file to another location.

Example: moveto file.txt /tmp/

chperm

chperm [file] [permissions]

Changes file permissions (octal format).

Example: chperm script.sh 755

file-stat

file-stat [file] [options]

Displays file statistics.

Options:
  • -p, --permissions: Show only permissions
  • -s, --size: Show only file size
  • --help: Display help information
Example: file-stat document.txt -s Output: 1.2 MB

Directory Operations

sd

sd [directory] [options]

Scans and displays directory contents.

Options:
  • -f, --full-path: Display full paths
  • -r, --recursive: Recursive scanning
  • --help: Display help information
Example: sd /home/user -r

create dir

create dir [directory_name]

Creates a new directory.

Example: create dir myfolder

copydir

copydir [source] [destination]

Copies a directory recursively.

Example: copydir /home/user /backup/

movedir

movedir [source] [destination]

Moves a directory to another location.

Example: movedir /old /new/location

deldir

deldir [directory]

Deletes a directory and all its contents.

Example: deldir temporary/

Process Commands

stop

stop [pid] [signal]

Sends a signal to a process. Default signal is 15 (SIGTERM).

Example: stop 1234 9 Output: Sends SIGKILL (9) to process 1234

lp

lp

Lists all running processes with their PIDs and names.

Example: lp Output: Displays process list
